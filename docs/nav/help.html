Инструкция по решению задачи


1

Создайте аккаунт в сети Rinkeby. Используйте для этого MetaMask

2

На faucet Rinkeby получите тестового ETH для решения задач

3

Нажмите на кнопку

4

Вам известен исходный код, найдите в нем уязвимость. Используйте для этого тестовую среду, советуем такие инструменты, как Remix IDE или Truffle Framework + Ganache.

5

Для того, чтобы решение засчиталось, вам нужно вызвать уязвимость на адресе, который вы получите после нажатия кнопки в п. 3.

Советуем вам делать это через Remix IDE с помощью MetaMask (At address - вызывайте функцию и передавайте туда полученный адрес)

7

Как только вы сделали нужные операции с контрактом в сети, нажмите на кнопку
